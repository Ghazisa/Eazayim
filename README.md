# مِعزَب الدعوات 💌

تطبيق ويب عربي (RTL) لأرشفة دعوات الزواج السعودية: ترفع صورة الدعوة بأي خط عربي، يمسحها الذكاء الاصطناعي، يستخرج المعلومات المميزة، ويعبّئ الحقول تلقائياً — ثم تُحفظ الدعوات وتُعرض في أرشيف أنيق مع عدّاد الأيام المتبقية.

## هيكل المشروع

```
mezab/
├── index.html        ← الهيكل (ابدأ من هنا)
├── css/
│   └── style.css     ← كل التنسيق والهوية البصرية
└── js/
    ├── config.js     ← الإعدادات ومفتاح API (عدّل هنا أولاً)
    ├── utils.js      ← توحيد الأرقام والتواريخ (الكود السعودي الموحد)
    ├── storage.js    ← الحفظ المحلي + النسخ الاحتياطي
    ├── ocr.js        ← محرك المسح والاستخراج
    └── app.js        ← منطق الواجهة وسير العملية
```

## التشغيل في VS Code

1. افتح مجلد `mezab` في Visual Studio Code.
2. **اضبط المفتاح**: افتح `js/config.js` وضع مفتاحك في `ANTHROPIC_API_KEY`
   - أنشئ المفتاح من: https://console.anthropic.com → API Keys
   - أو استخدم Gemini: غيّر `PROVIDER` إلى `"gemini"` وضع مفتاحاً من https://aistudio.google.com
3. ثبّت إضافة **Live Server** ثم زر يمين على `index.html` → **Open with Live Server**.
   (يعمل أيضاً بفتح `index.html` مباشرة في المتصفح، لكن Live Server أفضل)

## سير العملية

1. **إضافة دعوة** → ارفع صورة الدعوة (أو صوّرها بالجوال).
2. المحرك يضغط الصورة ويرسلها للنموذج الذي **يقرأ نصها الفعلي** بأي خط
   (ثلث، ديواني، نسخ، رقعة...) ويستخرج ٩ حقول.
3. الحقول **تتعبأ تلقائياً** — تراجعها وتصحح أي تفصيلة ثم **حفظ**.
4. الدعوة تُحفظ محلياً على جهازك وتبقى بعد إغلاق المتصفح والعودة.

## توحيد المخرجات (الكود السعودي الموحد)

- التاريخ الهجري واليوم والملاحظات: أرقام عربية شرقية (١٤٤٨).
- الهاتف والتاريخ الميلادي: أرقام إنجليزية (0501234567 / 2026-07-17).
- التوحيد يطبَّق تلقائياً على مخرجات المسح وعلى الإدخال اليدوي عند الحفظ.

## ملاحظات

- **النسخة الاحتياطية**: زر «حفظ نسخة احتياطية» ينزّل ملف JSON بكل دعواتك،
  و«استعادة نسخة» يعيدها على أي جهاز.
- **الحذف**: زر ✕ على كل بطاقة، أو «حذف الدعوة» داخل التفاصيل.
- المفتاح يبقى على جهازك فقط (يُرسل مباشرة للمزوّد من متصفحك). لا تشارك
  ملف `config.js` مع أحد، وإن نشرت المشروع علناً فالأسلم نقل الاتصال إلى
  خادم وسيط بسيط بدل وضع المفتاح في المتصفح.
