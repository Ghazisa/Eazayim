# مِعزَب الدعوات 💌

تطبيق ويب عربي (RTL) لأرشفة دعوات الزواج السعودية: ترفع صورة الدعوة بأي خط عربي، يمسحها الذكاء الاصطناعي، يستخرج المعلومات المميزة، ويعبّئ الحقول تلقائياً — ثم تُحفظ الدعوات وتُعرض في أرشيف أنيق مع عدّاد الأيام المتبقية.

## هيكل المشروع

```
mezab/
├── index.html        ← الهيكل (ابدأ من هنا)
├── css/
│   └── style.css     ← كل التنسيق والهوية البصرية
└── js/
    ├── config.js     ← الإعدادات ومفتاح API (عدّل هنا أولاً)
    ├── utils.js      ← توحيد الأرقام والتواريخ (الكود السعودي الموحد)
    ├── storage.js    ← الحفظ المحلي + النسخ الاحتياطي
    ├── ocr.js        ← محرك المسح والاستخراج
    └── app.js        ← منطق الواجهة وسير العملية
```

## التشغيل في VS Code

1. افتح مجلد `mezab` في Visual Studio Code.
2. **اضبط المفتاح**: افتح `js/config.js` وضع مفتاحك في `ANTHROPIC_API_KEY` أو `GEMINI_API_KEY` حسب المزود.
3. ثبّت إضافة **Live Server** ثم زر يمين على `index.html` → **Open with Live Server**.
